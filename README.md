rbg 
a Rev Bayes Generator

Goal
####

Python library to generate Rev Bayes syntax from python

Installation
############

    $ git clone https://github.com/mtholder/rbg
    $ virtualenv rbg-env
    $ source rbg-env/bin/activate
    $ cd rbg
    $ python setup.py develop

Usage
#####

    $ cd example
    $ python log_normal.py
    $ rb log_normal.Rev
